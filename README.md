# 🤖 HR AI Screening Agent

> Built by **Khushboo Manchanda** | CSPO Certified | Senior Product Owner turned AI Builder  
> Built with zero coding experience using Make.com + OpenAI

## What this does

This AI agent automatically screens job applicants and gives HR teams an instant, structured evaluation report — saving hours of manual resume review per hire.

**In under 5 seconds, it returns:**
- A score out of 10 across 5 dimensions
- Top 3 candidate strengths
- Top 2 gaps to probe in interviews
- A clear recommendation: SHORTLIST / MAYBE / REJECT
- A suggested interview question tailored to the candidate

## Sample output

CANDIDATE NAME: Khushboo Manchanda
ROLE APPLIED FOR: Senior Product Manager
TOTAL SCORE: 9.5/10
RECOMMENDATION: SHORTLIST

## How it works

Candidate submits resume → Webhook receives it → OpenAI evaluates → Report returned instantly
